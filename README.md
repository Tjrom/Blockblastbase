# BlockBlast

Смарт-контракт для Base Dev

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000) в браузере.

## Деплой на Vercel

Проект готов к деплою на Vercel. Просто подключите репозиторий GitHub к Vercel.
